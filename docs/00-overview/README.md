이 템플릿을 **초보 개발자분들, 1인 빌더분들**과 나누고 싶었어요.  
최대한 쉽게, 눈높이에 맞춰 설명하려고 노력했으니 천천히 따라와 주세요!

## 🤖 AI 코딩 툴 사용자라면?

**Cursor**, **Windsurf**, **Claude Code** 같은 AI 코딩 툴을 쓰시나요?

그렇다면 이 문서 하나만 AI에게 첨부하세요:

```
📎 docs/llm.md
```

> **"이 문서대로 설정 도와줘"** 라고만 하면,  
> AI가 단계별로 친절하게 안내해줄 거예요!

`llm.md`에는 AI가 따라야 할 모든 지침이 들어있어요:

- 어떤 순서로 진행해야 하는지
- 각 단계에서 어떤 문서를 참조해야 하는지
- 자주 나오는 에러와 해결책
- 친근하게 설명하라는 스타일 가이드

혼자 진행하셔도 되고, AI와 함께 진행하셔도 돼요.  
어느 쪽이든 **이 문서들은 따라하기 쉽게** 작성되어 있어요!

## 🎯 이 보일러플레이트는 뭔가요?

**Global SaaS Starter Kit**은 SaaS를 빠르게 만들 수 있는 템플릿이에요.

로그인, 결제, 이메일 같은 **모든 SaaS에 필요한 기능**이 이미 구현되어 있어서,  
여러분은 **핵심 기능 개발에만 집중**할 수 있어요.

## 🔧 사용하는 서비스들

### 1. Supabase (수파베이스)

**한 줄 설명**: Firebase의 오픈소스 대안. 데이터베이스 + 인증을 제공해요.

**왜 쓰나요?**

- Google, GitHub 로그인을 5분 만에 구현
- PostgreSQL 데이터베이스를 클릭 몇 번으로 생성
- 무료 플랜으로 시작 가능

**이 프로젝트에서 하는 일**:

- 🔐 **사용자 인증** (Google OAuth, Magic Link)
- 🗄️ **데이터 저장** (사용자 정보, 구독 상태)
- 🔒 **보안** (Row Level Security로 데이터 보호)

🔗 [Supabase 공식 사이트](https://supabase.com)

---

### 2. LemonSqueezy (레몬스퀴지)

**한 줄 설명**: 한국 사업자도 사용 가능한 글로벌 결제 서비스.

**왜 쓰나요?**

- **한국 사업자 지원** (Stripe와 달리 별도 법인 불필요)
- **Merchant of Record**: 세금, 부가세 처리를 대신해줌
- 구독 관리, 고객 포털 제공

**이 프로젝트에서 하는 일**:

- 💳 **결제 처리** (체크아웃 오버레이)
- 📅 **구독 관리** (갱신, 해지, 플랜 변경)
- 📧 **영수증 자동 발송**
- 🔔 **Webhook** (결제 이벤트를 서버로 알림)

🔗 [LemonSqueezy 공식 사이트](https://lemonsqueezy.com)

---

### 3. Resend (리센드)

**한 줄 설명**: 개발자 친화적인 이메일 발송 서비스.

**왜 쓰나요?**

- React 컴포넌트로 예쁜 이메일을 쉽게 제작
- 배달율이 높음 (스팸함 안 감)
- 무료 플랜: 월 3,000통

**이 프로젝트에서 하는 일**:

- 📬 **Welcome 이메일** 발송
- ⚠️ **결제 실패 알림** 발송
- 📝 **맞춤 이메일 템플릿** 제공

🔗 [Resend 공식 사이트](https://resend.com)

---

### 4. Vercel (버셀)

**한 줄 설명**: Next.js를 만든 회사의 호스팅 서비스.

**왜 쓰나요?**

- Git push만 하면 자동 배포
- HTTPS, CDN 모두 무료
- 서버리스라서 서버 관리 불필요

**이 프로젝트에서 하는 일**:

- 🚀 **웹사이트 호스팅**
- 🌍 **글로벌 CDN으로 빠른 로딩**
- 🔄 **자동 배포** (GitHub 연동)

🔗 [Vercel 공식 사이트](https://vercel.com)

## 📁 기술 스택 요약

| 분류           | 기술                     | 설명                |
| -------------- | ------------------------ | ------------------- |
| **프레임워크** | Next.js 16               | React + 서버 렌더링 |

| **다국어**     | next-intl                | 한국어/영어 지원    |
