## 🏷️ 메타데이터가 뭔가요?

검색 엔진이나 SNS가 "이 사이트는 뭐하는 곳이야?"라고 물어볼 때 대답해주는 정보예요.

### 1. 기본 메타데이터

| 항목          | 하는 일                    | 예시                        |
| ------------- | -------------------------- | --------------------------- |
| `title`       | 검색 결과 제목             | "TaskFlow - 팀 협업 도구"   |
| `description` | 검색 결과 설명 (155자까지) | "팀과 함께 일하세요..."     |
| `keywords`    | 검색에 걸리는 키워드       | "SaaS, 협업, 프로젝트 관리" |

### 2. Open Graph (카카오톡, 페이스북 공유용)

링크를 공유했을 때 예쁘게 미리보기가 뜨게 해주는 설정이에요.

| 항목   | 권장 사항      |
| ------ | -------------- |
| 제목   | 60자 이내      |
| 설명   | 155자 이내     |
| 이미지 | **1200x630px** |

---

## ⚙️ 설정 방법

### Step 1: layout.tsx 수정하기

`src/app/[locale]/layout.tsx` 파일을 열어서 metadata 부분을 수정하세요:

```tsx
export const metadata: Metadata = {
  title: {
    default: "내 서비스 이름",
    template: "%s | 서비스 이름",
  },
  description: "내 서비스를 한 줄로 소개하세요.",

  openGraph: {
    title: "서비스 이름",
    description: "카카오톡/페이스북에 공유될 때 보일 설명",
    images: [{ url: "/og-image.png", width: 1200, height: 630 }],
  },
};
```

### Step 2: OG 이미지 바꾸기

1. **1200x630px** 크기로 이미지를 만들어요
2. `public/og-image.png`로 저장하세요
3. 카카오톡이나 슬랙에 링크 보내서 미리보기가 잘 뜨는지 확인!

### Step 3: Favicon 바꾸기

1. 로고를 `.ico` 형식으로 변환하세요
2. `public/favicon.ico`로 저장하면 끝!
