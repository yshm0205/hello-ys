# SaaS MVP에 꼭 필요한 5가지 기능

MVP를 만들고 계신가요? 반드시 필요한 기능들을 알려드릴게요.

## 1. 인증

직접 인증을 만들지 마세요. Supabase나 Clerk를 사용하세요:

- 소셜 로그인 (Google, GitHub)
- 매직 링크 이메일
- 세션 관리

## 2. 결제

첫날부터 매출이 중요해요. Stripe가 제공하는 것:

- 구독 관리
- 고객 포털
- 자동 청구서

## 3. 랜딩 페이지

랜딩 페이지에서 답해야 할 질문:

- 어떤 문제를 해결하나요?
- 어떻게 해결하나요?
- 비용은 얼마인가요?

## 4. 대시보드

사용자가 계정을 관리할 공간이 필요해요:

- 프로필 설정
- 구독 상태
- 핵심 기능

## 5. 이메일 알림

사용자 참여를 유지하세요:

- 환영 이메일
- 결제 영수증
- 중요 업데이트

## 빠르게 런칭하세요

오버 엔지니어링하지 마세요. 이 5가지 기능이면 아이디어를 검증할 수 있어요. 일단 출시하세요!
